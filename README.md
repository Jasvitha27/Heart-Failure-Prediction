# Heart Failure Prediction using Machine Learning

## Project Overview

This project predicts the likelihood of heart failure using clinical patient records. It involves data preprocessing, exploratory data analysis, feature scaling, and the implementation of Machine Learning and Deep Learning models to classify patients based on their risk of heart failure.

---

## Objectives

- Understand the clinical dataset and its features.
- Perform data cleaning and preprocessing.
- Analyze feature distributions and relationships.
- Train Machine Learning and Deep Learning models.
- Compare model performance for heart failure prediction.

## Features Used

The dataset contains important clinical attributes such as:

- Age
- Anaemia
- Creatinine Phosphokinase (CPK)
- Diabetes
- Ejection Fraction
- High Blood Pressure
- Platelets
- Serum Creatinine
- Serum Sodium
- Sex
- Smoking
- Time

Target Variable:

- Death Event (Heart Failure)

---

## Dataset

This project uses the **Heart Failure Clinical Records Dataset** available on Kaggle.

Dataset Link:

https://www.kaggle.com/datasets/andrewmvd/heart-failure-clinical-data

The dataset is included in this repository for educational purposes.
